# JRMicro Frontend

### Install Dependencies

```
cd jrcommon && yarn
cd jrprofile && yarn
cd jrmain && yarn
```

### Run JRCommon

```
cd jrcommon && yarn dev
```

### Run JRProfile

```
cd jrprofile && yarn build && yarn serve
```

### Run JRMain

```
cd jrmain && yarn dev
```
